# GPP-25-26-Level-Design

A template project for the second term of GPP 25-26. This will provide a set of mechanics similar to what would be done for the first assignment. The goal is to then build and design a level around them using game design and level design theory.



IMPORTANT DISCLAIMER: I am really, really, really bad at platformer games. This level is playable in 4 minutes FOR ME and the correct level of challenging FOR ME, but someone better at platformers might complete this faster than me, or find it really easy. I thought it best to program around my own skill level because I am the one who is constantly testing it during development.
